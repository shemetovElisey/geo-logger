# GeoLogger Demo App

Демонстрационное iOS приложение для записи и воспроизведения геопозиции с использованием GeoLogger SDK.

## Настройка проекта

### 1. Добавление зависимости GeoLogger

1. Откройте проект `Demo.xcodeproj` в Xcode
2. Выберите проект в навигаторе (самый верхний элемент)
3. Выберите target "Demo"
4. Перейдите на вкладку "Package Dependencies"
5. Нажмите "+" для добавления зависимости
6. Выберите "Add Local..." и укажите путь к корню репозитория geo-logger:
   ```
   ../../../
   ```
   (относительно папки Examples/Demo/)
7. Или если SDK опубликован, добавьте через URL репозитория

### 2. Проверка Info.plist

Убедитесь, что в настройках проекта добавлены разрешения на геолокацию:
- `NSLocationWhenInUseUsageDescription`
- `NSLocationAlwaysAndWhenInUseUsageDescription`

Если используется встроенный Info.plist в настройках проекта, добавьте эти ключи там.

### 3. Сборка и запуск

1. Выберите симулятор или устройство
2. Нажмите Cmd+R для сборки и запуска
3. При первом запуске разрешите доступ к геолокации

## Использование

### Запись геопозиции

1. Нажмите "Start Recording"
2. Разрешите доступ к геолокации при первом использовании
3. Приложение начнет записывать ваше местоположение
4. Нажмите "Stop Recording" для завершения

### Воспроизведение

1. Выберите запись из списка (нажмите на неё)
2. Настройте скорость воспроизведения слайдером (0.5x - 10x)
3. Нажмите "Start Replay"
4. Нажмите "Stop Replay" для остановки

### Управление записями

- **Выбор**: нажмите на запись в списке
- **Удаление**: нажмите на иконку корзины
- **Экспорт**: нажмите на иконку Share для экспорта через Share Sheet

## Структура файлов

- `DemoApp.swift` - точка входа приложения
- `ContentView.swift` - главный экран с UI
- `LocationViewModel.swift` - ViewModel с логикой работы с GeoLogger
- `Info.plist` - конфигурация с разрешениями

## Примечания

- Записи сохраняются в `Documents/GeoLogger/`
- Для тестирования на симуляторе используйте функции симуляции местоположения в Xcode (Debug → Location)
